========================================
  MaggiBank - Mobile App Requirements
========================================

PROJECT OVERVIEW
----------------
MaggiBank is a React Native mobile banking application built with Expo.
App name   : MaggiBank
Version    : 1.0.0
Slug       : maggibank
Orientation: Portrait only
Theme      : Dark


SYSTEM REQUIREMENTS
-------------------
- Node.js  : v18 or higher (LTS recommended)
- npm      : v9 or higher (comes with Node.js)
- Git      : any recent version


TOOLS & CLI
-----------
- Expo CLI : install globally with  -->  npm install -g expo-cli
             or use npx (no global install needed)
- EAS CLI  : optional, for building/submitting to stores
             install with  -->  npm install -g eas-cli


PLATFORM REQUIREMENTS
---------------------
To run on Android:
  - Android Studio installed with Android SDK
  - An Android emulator configured, OR a physical Android device
    with USB debugging enabled
  - ANDROID_HOME environment variable set correctly

To run on iOS (macOS only):
  - Xcode installed (from the Mac App Store)
  - iOS Simulator or a physical iPhone/iPad

To run on Web:
  - Any modern browser (Chrome, Firefox, Edge, Safari)


ENVIRONMENT SETUP
-----------------
1. Copy or create a .env file in the root of this folder (mobile-app/).
   It must contain the following variables:

   SUPABASE_URL=<your_supabase_project_url>
   SUPABASE_ANON_KEY=<your_supabase_anon_key>

2. The app.json file contains an "extra" field with the analytics server URL:
   "analyticsUrl": "http://<your_local_ip>:8000"
   Update this IP to match the machine running the analytics/backend server.


INSTALLATION
------------
1. Navigate to the mobile-app folder:
     cd mobile-app

2. Install dependencies:
     npm install

3. Start the development server:
     npm start          (opens Expo Dev Tools in the browser)
     npm run android    (starts on Android emulator/device)
     npm run ios        (starts on iOS simulator/device — macOS only)
     npm run web        (starts in the browser)
